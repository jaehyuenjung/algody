# ✏ 회고록

## 문제
[백준 - 뱀](https://www.acmicpc.net/problem/3190)

## 후기
```
이번 문제는 뱀의 특성(?)을 잘 이해하면 효율적으로 문제를 해결할 수 있다.

뱀의 머리가 지나간 곳에는 반드시 꼬리가 지나간다는 특성을 이용하면 몸통에 대한 움직임을 고려할 필요가 없다.

하지만 나는 이러한 점을 고려하지 못하고 그대로 머리, 몸통, 꼬리의 움직임을 모두 구현하였고
비록 문제를 해결했지만, 구현하는 과정이 복잡해서 문제 푸는데 시간이 좀 소요됐다.

문제를 해결하고 나서야 뱀의 특성을 깨달아서 효율적인 풀이 방식으로 문제를 풀 수 있었지만,
좀 더 빨리 알았더라면 시간을 덜 잡아먹지 않았을까 하는 아쉬움이 있었다.

구현 문제의 핵심은 지문의 있는 내용을 그대로 코드화하는 것보다는 
어떻게 하면 요구되는 모든 과정을 쉽게 해결할 수 있는 방법을 찾는 것이라는 것을 알 수 있었다.
다음 구현 문제를 풀 때는 이점을 적용할 수 있도록 해야겠다!
```
