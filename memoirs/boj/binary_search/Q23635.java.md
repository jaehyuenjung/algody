# ✏ 회고록

## 문제
[백준 - 산타로부터의 선물](https://www.acmicpc.net/problem/23635)

## 후기
```
난이도 있는 누적합 알고리즘 문제를 찾던 중 난이도 0인 문제가 있어 호기심에 풀게 되었다.

누적합 알고리즘을 그대로 적용해서 풀 수 있는 문제라 생각하고 도전했지만,
시간 초과라는 결과가 나와서 당황스러웠다.

‘난이도 0이라서 쉽게 풀 수 있겠지’라고 생각하였는데, 알고 보니 난이도를 매길 수 없는 문제였다.

풀이 방법이 도저히 떠오르지 않아서 문제의 알고리즘을 보게 되었고 이중 탐색이라는 알고리즘을 알게 되었다.
마침 탐색 시간을 줄일 필요가 있어서 이중 탐색 알고리즘을 적용하니 문제를 풀 수 있었다.

이때까지 문제의 힌트를 얻기 위해 문제의 조건, 알고리즘, 내용에 집중하였다.
그래서 처음 풀이를 할 때도 문제를 그대로 구현하면 된다고 생각하였다.
하지만 이번 문제의 힌트는 출력값의 범위에 있었다.
입력과 출력값의 범위도 문제의 핵심이 될 수 있다는 것을 알게 해주었다.

앞으로 문제를 풀 때 입력과 출력값 범위에 유의하고 
새롭게 배운 이중 탐색 알고리즘에 익숙해지기 위해 다양한 문제를 풀어봐야겠다!
```
