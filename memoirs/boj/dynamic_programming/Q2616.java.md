# ✏ 회고록

## 문제
[백준 - 소형기관차](https://www.acmicpc.net/problem/2616)

## 후기
```
이때까지 풀어봤던 누저합 알고리즘 문제는 각 부분이 연속적인 합으로 되어있고 그 부분도 연속적으로 배치된 것을 구하는 문제였다.
예를 들어 1 ~ 4로 이루어진 부분이면 다음 부분은 무조건 5로 시작해야된다.

하지만 이번 문제는 각 부분이 부분합으로 이루어진 문제였다.

어떻게 풀면 좋을지 생각하다가 모든 경우의 수를 셀 수 밖에 없으므로 DFS 알고리즘을 적용하면 되지 않을까 생각해서 제출했지만,
시간초과 결과가 나와서 막막했다. 

결국 다른 풀이 방법이 도저히 생각나지 않아 알고리즘 유형을 보았고 다이내믹 프로그래밍 알고리즘을 알게 되었다.

다이내믹 프로그래밍 개념을 파악한뒤 문제로 돌아와서 풀려고 했으나,
점화식을 어떻게 세워야하는지 막막했다...

결국 다른 사람들의 풀이 보아서 풀게 되었다.

이번 문제를 통해 다이내믹 프로그래밍 알고리즘 개념을 알 수 있었다.
다이내믹 프로그래밍 알고리즘의 핵심은 점화식을 세우는건데,
이 부분에서 대해서 많은 부족함을 느낄 수 있었다.
비슷한 유형의 문제를 풀면서 감을 익힐 수 있도록 해야겠다!
```
