# ✏ 회고록

## 문제
[백준 - 꿀 따기](https://www.acmicpc.net/problem/21758)

## 후기
```
문제를 보자마자 최대한 많은 꿀을 모이기 위해서는 벌통의 위치가 한정적이기 때문에
그리디 알고리즘 문제라고 생각했다.

하지만 온전히 나의 감으로 그리디 알고리즘 문제라고 판단할 수 있는 자신감이 없었기 때문에
계속 계속해서 방법을 찾았다.

수학, DFS 알고리즘 등등 여러 가지를 생각해 보았지만,
해결법을 찾지 못했고 결국 다른 사람들의 풀이를 보게 되었다..

그리디 + 다이내믹 알고리즘을 이용한 풀이를 보면서 약간의 아쉬움이 남았다.
처음 느낌대로 계속 생각했으면 풀 수 있는 문제인데,
그러지 못한 것에 대한 후회가 남았다...

이번 문제를 통해 알고리즘 문제를 푸는 것에 대한 의미를 되새길 수 있었다.
이전까지는 문제를 통해 학습하는 것이 아닌 문제를 덜 틀리면서 빠르게 맞는 것을 목표로 하지 않았나 생각이 든다.
지금은 시험이 아니라 배우는 과정이라는 것을 명심하면서 더 많이 틀리고 배울 수 있는 마음가짐을 가져야겠다!
```
