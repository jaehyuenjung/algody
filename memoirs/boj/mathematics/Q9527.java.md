# ✏ 회고록

## 문제
[백준 - 1의 개수 세기](https://www.acmicpc.net/problem/9527)

## 후기
```
이번 문제는 짧은 설명으로 구성되어 있어 이해하는 데 어려움은 없었지만,
어떻게 풀어야 할지 난감했다.

입력과 출력 범위를 통해 수학 알고리즘을 이용해야겠다는 것은 바로 파악했지만,
어떻게 적용할지가 어려웠다.

공책에 공식을 도출하기 위해 많은 시도를 하였지만,
별다른 소득이 없었다..

결국 다른 풀이를 살펴보게 되었다.
하지만 이마저도 단순히 코드만으로는 왜 그런 공식이 도출되었는지 이해하기 어려웠다.

풀이를 이해하기 위해 여러 번의 검색을 하였고 백준 님의 1019문제 풀이를 알게 되었다.
거기에는 공식이 도출되는 과정이 이해하기 쉽게 설명되어있었다.

풀이 설명을 보면서 색다른 아이디어로 문제를 쉽게 해결되는 것이 신기하고 놀라웠다.

비록 문제는 달랐지만, 아이디어는 똑같이 적용할 수 있다는 느낌이 들었고
바로 적용하니 통과할 수 있었다.

수학 알고리즘은 왠지 다른 알고리즘과 달리 수학 공식을 이용하기 때문에
얼마나 수학에 대해 더 많이 알고 있는가에 풀이 여부가 달라지는 것 같다.
앞으로 수학 알고리즘을 좀 더 많이 풀어서 지식을 쌓아나가야겠다!
```
