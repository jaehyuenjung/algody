# ✏ 회고록

## 문제
[백준 - 수들의 합 4](https://www.acmicpc.net/problem/2015)

## 후기
```
이번 문제는 부분합의 관계식을 잘 이해하고 있는가에 따라 풀이 여부가 갈릴 수 있는 문제인 것 같다.

처음 문제를 읽었을 때 "그대로 부분합 알고리즘을 구현하면 해결할 수 있지 않나?"라고 생각했지만
입력 조건을 보자마자 시간초과가 발생할 수밖에 없도록 설정했다는 것을 알았다.

그래도 혹시 몰라 제출하였지만, 역시 시간초과가 발생하면서 실패하였다..

아무리 생각해도 문제를 해결하려면 모든 부분합을 구해야 하는데 그 부분에서 시간이 오래 걸린다는 것이 이상했다.
이때까지 풀이 경험을 생각해 보았을 때 수학 알고리즘을 적용해야 할 것 같은 느낌이 들었다.

그래서 모든 부분합을 구하는 공식을 자세히 살펴보았더니 쉽게 풀 수 방법을 찾았고 문제를 해결할 수 있었다.

이번 문제를 통해 다시금 문제가 시간초과가 발생할 수밖에 없도록 설정되었을 때
수학 알고리즘을 고려해보는 노하우를 얻을 수 있었다.
다른 문제도 이러한 상황이 발생하였을 때 적용할 수 있도록 해야겠다!
```
