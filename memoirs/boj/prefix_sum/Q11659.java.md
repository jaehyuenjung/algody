# ✏ 회고록

## 문제
[백준 - 구간 합 구하기 4](https://www.acmicpc.net/problem/11659)

## 후기
```
이번 문제는 누적합 알고리즘을 그대로 적용하면 풀 수 있는 문제였다.
그래서 어렵지 않게 문제를 해결할 수 있었다.

좀 더 좋은 성능을 낼 수 있는 코드가 있는지 궁금해서 다른 사람들의 풀이를 살펴보았다.
...? 무려 5배 가깝게 시간을 단축시킨 코드들이 있었다.
어떻게 하면 시간을 줄일 수 있는지 궁금해서 코드들을 분석하였다.

풀이 방식은 대부분 비슷했지만, 입력과 출력하는 과정에서 차이가 있었다는 것을 알았다.

먼저 숫자를 입력 받아올 때는 BufferedReader 클래스를 사용하여 라인 단위로 읽는 것 보다 바이트 단위로 처리하는 것이 더 빠르다.
이것은 문자에서 숫자로 변환하는 과정을 생략할 수 있어서 시간을 절약할 수 있다는 것을 알 수 있었다.

다음으로 출력을 할 때는 시스템 출력 명령어를 남발하기 보다는 문자열로 한번에 저장해서 출력하는 것이 더 효율적이다.
시스템 출력은 자원을 많이 소모하므로 지양하는 것이 좋다. 

문제 풀고나서 자바 I/O에 대한 지식이 부족하다는 것을 알 수 있었다.
기본서를 참고하여 보완해 나갈 수 있도록 해야겠다!
```
